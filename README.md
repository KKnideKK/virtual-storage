# virtual-storage
请求式分页储存管理系统。将输入的逻辑地址计算出其页号，通过页号查找快表，若快表未命中则查找页表，页表未命中则缺页中断。通过先进先出置换算法或最少使用置换算法置换页面。找到对应物理块号后乘以页面大小加上偏移量得到物理地址，并输出页表、快表状态。在计算出最后一个物理地址后输出缺页率。
